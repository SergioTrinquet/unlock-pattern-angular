# Projet Unlock Pattern

Ce projet est une application web développée avec Angular (version 20.3.0), pour implémenter une fonctionnalité de "schéma de déverrouillage" (unlock pattern), comme celles que l'on trouve sur les appareils mobiles.

  Bibliothèques utilisées :
   * RxJS (pour la programmation réactive),
   * js-confetti (pour des effets visuels, potentiellement lors d'un déverrouillage réussi).
